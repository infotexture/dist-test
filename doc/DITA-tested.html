Tested platforms and tools

See which tools and platforms have been used in testing the DITA processing system.

The DITA processing system has been tested against the following platforms and tools:

Tested OS: Windows XP, SLES 10
Tested XSLT processor: Xalan-J 2.6, Saxon 6.5, Saxon 9
Note: XSLT 2.0 standard is not officially supported yet, due to the reliance by some users on Xalan.
Tested JDK: IBM 1.5, 1.6; SUN 1.5, 1.6
Tested Ant: Ant 1.7.1
