Installing XALAN XSLT Processor

You can alternatively use the XALAN SXLT Processor to import stylesheets and queries into an XML schema, validate XML data against a schema, and select the elements and attributes in your XML documents according to their schema-defined type.

  1. Enter the URL:http://archive.apache.org/dist/xml/xalan-j/
  2. From SAXON: The Xalan Processor page, scroll to find the heading xalan-j_2_6_0-bin.zip. Click to downlad.
  3. Save and unzip the xalan-j_2_6_0-bin.zip file to C:\ directory as xalan.
  4. Set the CLASSPATH environment variable for following .jar files: xalan.jar file and the xercesImpl.jar file
